Development
===========

Development for this package takes place in its `GitHub repository
<https://github.com/karlotness/adrt/>`__. From there you can request
support, report issues, and submit pull requests.

`Ask a question <https://github.com/karlotness/adrt/discussions>`__
  for general discussion and questions

`File a bug <https://github.com/karlotness/adrt/issues>`__
  for problems with the software

`Submit changes <https://github.com/karlotness/adrt/pulls>`__
  to contribute changes to the code or documentation. See
  :doc:`develop.contribute` for more information on processes
